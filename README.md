# 🏠️ Airbnb Clone

A full-stack web application inspired by Airbnb where users can list properties, explore listings, and share reviews with ratings.

This project replicates the core features of a property rental marketplace including listing management, image uploads, authentication, and location geocoding.

---

## 🌐 Live Demo

👉 [Live Website](https://airbnb-clone-c5r4.onrender.com)

---

## ✨ Features

🏠 **Property Listings**

- Create new property listings
- Edit existing listings
- Delete listings

📸 **Image Uploads**

- Upload listing images using Multer
- Images stored in Cloudinary

⭐ **Reviews & Ratings**

- Users can comment on listings
- Star rating system (1–5 stars)
- Users can delete their own comments

📍 **Location Geocoding**

- Converts location text into geographic coordinates
- Powered by OpenStreetMap Nominatim API

🔐 **Authentication**

- Secure user authentication using Passport

## 🛠️ Tech Stack

### 🎨 Frontend

- HTML
- CSS
- JavaScript
- Bootstrap
- EJS (templating)

### ⚙️ Backend

- Node.js
- Express.js
- MongoDB
- Mongoose

### 🔧 Libraries & Services

- Passport (authentication)
- Joi (validation)
- Multer (file uploads)
- Cloudinary (image storage)
- connect-mongo (session storage)
- OpenStreetMap Nominatim API (geocoding)
